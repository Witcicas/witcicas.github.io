嘗試在Win上搭建L4D2服務器

发表于 2021-09-07 分类于 游戏 Disqus：

最近在Ali云上搞了一台win服務器，設置失誤QWQ 配置如下

CPU&内存 1核 2 GiB 100%性能
操作系统 Windows Server 2019 数据中心版 64位中文版
镜像ID win2019_1809_x64_dtc_zh-cn_40G_alibase_20210715.vhd

雖然操作熟悉，但打開時發現居然比垃圾站的奔騰神教還磨蹭，流暢級別可以說是混沌級了

開機>滴>4分鐘>滴>載入用戶>滴>環境設置…10分鐘

拿來挖門羅只有10H/s，於是決定設Gaming服務器了

然后解压缩放到自己設立的服務器文件夾

接着在文件夾任意新建*.bat格式

接着用記事本打開輸入

start steamcmd.exe +login anonymous +force_install_dir l4d2_sv +app_update 222860 validate +quit

保存後繼續雙擊，開始自動下載所需的問價

等待完成

完成后開始

進入l4d2_sv文件夾，打开srcds.exe

自定義設置完

用公網IP登録~

開始Gaming！
